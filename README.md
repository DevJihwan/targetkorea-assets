# TARGETKOREA Assets Repository

TARGETKOREA 리포트 자동화 시스템에서 사용되는 로고 및 에셋을 관리하는 레포지토리입니다.

## 📁 구조

```
targetkorea-assets/
├── logos/              # 플랫폼 로고 이미지
│   ├── kb-card.png
│   ├── nice.png
│   ├── lpoint.png
│   └── ...
└── README.md
```

## 🖼️ 로고 사용법

### 직접 URL 사용 (raw.githubusercontent.com)

```
https://raw.githubusercontent.com/DevJihwan/targetkorea-assets/main/logos/kb-card.png
```

### HTML에서 사용

```html
<img src="https://raw.githubusercontent.com/DevJihwan/targetkorea-assets/main/logos/kb-card.png" alt="KB Card">
```

- 확장자: .png (권장)

- `kb-card.png`
- `sk-telecom.png`
